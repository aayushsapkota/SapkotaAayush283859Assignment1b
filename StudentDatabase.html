Use of Local Storage

In this section, we look how we can use local storage to save, retrieve,reset and also change and manupulate the data using javascript.

Notes to use localStorage

Some browsers many not have localstorage or are not compatible, as it is must for using localstorage, in that case the button below shows whether your browser is compatible to localStorge and if it has local storage.

The following rules have to be followed to access anf edit the data from the database:

  • The database has validation checks. therefore, it will not save unless all requirements are met
  • Change And Delete Element displays a box, so that, data can be changed based on their indexes.
  • The indexes are based on the 5 data fields so, Name is given the first place, Date is given the second place and show on. The pattern is repeated every 5 element so, A object has indexes to it
  • For example, to access/edit email in second object in list, the index starts at 0 and email is in third position of second object so, index will be 7.
